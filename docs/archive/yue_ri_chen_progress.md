## 技术实现细节

### 月令旺衰计算规则

根据五行生克规则计算各地支在当月的旺衰状态：

| 状态 | 条件 |
|------|------|
| 旺 | 地支与月建相同（五行同我） |
| 相 | 地支五行生月建五行（生我） |
| 休 | 月建五行生地支五行（我生） |
| 囚 | 地支五行克月建五行（克我） |
| 死 | 月建五行克地支五行（我克） |

### 月破判断

- 当日辰地支与月建地支相冲时为月破
- 相冲关系：子午、丑未、寅申、卯酉、辰戌、巳亥

### 旬空计算

- 根据日辰干支计算旬中空亡的地支
- 六十甲子每十日一轮回，空亡两位地支
